# ADR 0007 — Stable IDs, display aliases

**Status:** accepted

**Decision:** typed ULIDs identify projects, foundations, tracks, iterations, runs, artifacts and events. Track letters, iteration numbers, filenames and branch names are display aliases.

**Consequences:** collaboration cannot collide through counters; aliases may change without breaking lineage.

**Rejected:** path-, branch-, or counter-derived identity.

**Revisit when:** never.
