# ADR 0014 — Non-sensitive macOS/Linux v1

**Status:** accepted

**Decision:** public v1 supports public/non-sensitive computational work and is release-blocked on macOS/Linux.

**Consequences:** sensitive/private modes, compliance claims and Windows release gating remain separate programs. Security boundaries are designed now but not marketed early.

**Rejected:** implying a private endpoint alone enables sensitive use; all-platform launch before the vertical slice is stable.

**Revisit when:** the deferred security readiness and Windows matrices pass their independent gates.
