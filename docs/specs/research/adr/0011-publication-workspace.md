# ADR 0011 — Publication is an optional workspace

**Status:** accepted

**Decision:** a project can enable Publications at scaffold or later. Publications consume approved claims/evidence and never change scientific lifecycle state.

**Consequences:** no mutually exclusive standard/paper-driven project mode. Cross-project publications require later verified imports.

**Rejected:** always scaffolding paper files; per-study paper mode that changes research gates.

**Revisit when:** publication workflows need domain-specific editorial profiles.
