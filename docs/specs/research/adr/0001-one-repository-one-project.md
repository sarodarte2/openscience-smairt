# ADR 0001 — One repository is one Research Project

**Status:** accepted

**Decision:** one Git repository represents one Research Project and primary governance boundary. Parallel approaches are tracks; iterations and runs live below tracks.

**Consequences:** independent objectives use new repositories. Cross-project publication aggregation is deferred. Project and repository identity must remain stable across worktrees.

**Rejected:** multiple unrelated studies per repository; branch-only scientific identity.

**Revisit when:** a verified cross-project workspace can preserve independent governance and provenance.
